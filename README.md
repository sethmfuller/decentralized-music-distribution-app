# Decentralized Music Rooms
[View the project proposal](ProjectProposal%20(Detailed).pdf)

## Project setup
```
npm install
```

### Compile and hot-reload for development
```
npm run serve
```

### Compile and minify for production
```
npm run build
```

### Run tests
```
npm run test
```

### Lint and fix files
```
npm run lint
```

### Run end-to-end tests
```
npm run test:e2e
```

### Run unit tests
```
npm run test:unit
```
